# Skillgun Web Automation - Selenium + Pytest (POM)

Automated testing project for Skillgun web application using
Selenium WebDriver and Pytest with Page Object Model (POM) design pattern.

## Modules Tested
- Login (mobile, email, password)
- Dashboard navigation
- Placement page
- Profile settings
- Edit contact details

## Tech Stack
- Python
- Selenium WebDriver
- Pytest
- Page Object Model (POM)
- ChromeDriver, WebDriver Manager

## How to Run
pip install -r requirements.txt
pytest test/test_skillgun.py
